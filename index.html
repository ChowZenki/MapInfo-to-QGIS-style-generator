MapInfo-to-QGIS-style-generator by NathanW2

Generates a QGIS qml file containing the closest match for a MapInfo style.

MapInfo to QGIS style converter

Dependencies

Qt QGIS MapInfo

Install

Just copy the python file to anywhere you want and: python mapinfoToQgis.py my.TAB out.qml -c FittingType --UseMapInfo
